Our Game

Munchy Garden is an offline educational game designed to encourage children about healthy eating habits in a fun and interactive way. The game operates entirely within the Android operating system and does not require any user login or account creation.

Developed by Vegerush, Highschool Students of Thursina International Islamic Boarding School.

Information We Collect

All data is stored internally within the Android operating system. The application runs entirely within the operating system of the Android smartphone device which is managed internally. Therefore, we do not collect any personal user data, and no user data is stored in any external database or cloud server.

How We Use Your Information

As there is no personal data collection, we do not use or process any personal information. Any data generated during gameplay (such as scores or progress) is stored locally on the device and is used solely to enhance the user experience within the app.
